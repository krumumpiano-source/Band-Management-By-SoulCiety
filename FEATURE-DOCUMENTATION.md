# 📖 BandThai — เอกสารฟีเจอร์ทั้งหมด (สำหรับเขียน User Manual)

> **ชื่อแอป**: BandThai (🎵 ระบบจัดการวงดนตรี)  
> **ประเภท**: Progressive Web App (PWA) — ติดตั้งบนหน้าจอ Home ได้  
> **Backend**: Supabase (PostgreSQL)  
> **Hosting**: GitHub Pages  
> **ภาษา**: ไทย (หลัก) + i18n support  
> **Monetization**: Google AdSense + แผนสมาชิก (Free / Lite / Pro)  
> **การชำระเงิน**: Omise (บัตรเดบิต/เครดิต)  
> **AI**: Gemini API สำหรับ AI Auto-Tag เพลง  
> **External APIs**: iTunes API, Spotify API (ค้นหาข้อมูลเพลง)

## 1. ระบบล็อกอิน & สมัครสมาชิก

### 1.1 หน้าล็อกอิน (`index.html`)
- **ฟิลด์**: อีเมล, รหัสผ่าน (มีปุ่มสลับแสดง/ซ่อนรหัสผ่าน)
- **ปุ่ม**: `เข้าสู่ระบบ` (gradient สีทอง)
- **ลิงก์ด้านล่าง**:
  - สมัครสมาชิก → `register.html`
  - ขอสร้างวงใหม่ → `create-band.html`
  - ลืมรหัสผ่าน → `forgot-password.html`
  - ลืมอีเมล → `forgot-email.html`
  - ข้อกำหนดการใช้งาน → `terms.html`
- **ฟีเจอร์พิเศษ**: แสดง Banner เตือนหากเปิดใน In-App Browser (แนะนำให้เปิดใน Safari/Chrome)
- **ธีม**: พื้นหลังมืด (#0a0a15) สไตล์ Premium

### 1.2 สมัครสมาชิก (`register.html`)
- **วิซาร์ด 3 ขั้นตอน** พร้อมจุด indicator:
  - **ขั้นที่ 1** — กรอกรหัสประจำวง (6 หลัก): ระบบจะตรวจสอบกับ API แล้วแสดงชื่อวง
  - **ขั้นที่ 2** — ข้อมูลส่วนตัว:
    - คำนำหน้า (นาย/นาง/นางสาว/ด.ช./ด.ญ./ไม่ระบุ)
    - ชื่อ, นามสกุล, ชื่อเล่น
    - ตำแหน่ง/เครื่องดนตรี (12 ตัวเลือก): ร้องนำ, ร้องประสาน, กีตาร์ไฟฟ้า, กีตาร์โปร่ง, กีตาร์เบส, คีย์บอร์ด, กลอง, เพอร์คัสชัน, ทรัมเปต, แซกโซโฟน, ไวโอลิน, อื่นๆ
  - **ขั้นที่ 3** — สร้างบัญชี: อีเมล, รหัสผ่าน (8+ ตัวอักษร), ยืนยันรหัสผ่าน, แสดงสรุปก่อนกดยืนยัน
- **รองรับ URL parameter**: `?invite=XXXXXX` เพื่อเติมรหัสวงอัตโนมัติ

### 1.3 ลืมรหัสผ่าน / ลืมอีเมล / รีเซ็ตรหัสผ่าน
- `forgot-password.html` — กรอกอีเมลเพื่อรับลิงก์รีเซ็ต
- `forgot-email.html` — ค้นหาอีเมลจากชื่อเล่น/เบอร์โทร
- `reset-password.html` — ตั้งรหัสผ่านใหม่

## 3. คลังเพลง (Song Library)

**ไฟล์**: `songs.html` | **สิทธิ์**: ทุก Role

- **Layout**: Grid 3 คอลัมน์แสดงเพลง
- **ฟีเจอร์**:
  - 🔍 ค้นหาเพลง (แบบ real-time) พร้อมปุ่มล้าง (×)
  - 🏷️ Filters grid (หลายตัวเลือก auto-fit): กรองตามคีย์, แนวเพลง, นักร้อง, อารมณ์, ยุค ฯลฯ
  - แต่ละเพลงแสดง: ชื่อเพลง, ศิลปิน, คีย์, แนวเพลง, ประเภทนักร้อง (ชาย/หญิง/คู่)
- **การแสดงคีย์**: ใช้ `js/key-utils.js` สำหรับแสดงคีย์ดนตรี (C, 1#, 2b ฯลฯ)
- **ปุ่ม**: เพิ่มเพลง → `add-song.html`
- **โหมดเลือก (Select Mode)**: เข้าผ่าน `?selectMode=1` สำหรับเลือกหลายเพลงจากคลังกลางพร้อมกัน

## 4. เพิ่ม/แก้ไขเพลง

### 4.1 เพิ่มเพลง (`add-song.html`)
- **ฟิลด์**:
  - ชื่อเพลง * (required)
  - ศิลปิน / วงดนตรีต้นฉบับ (พร้อม autocomplete จาก master list)
  - คีย์ (C/Am, 1#-7#, 1b-7b)
  - BPM (0-300)
  - ยุคของเพลง: 80s, 90s, 2000s, 2010s, 2020s
  - อารมณ์เพลง: มัน/สนุก, หวาน/โรแมนติก, เศร้า/อกหัก, นิ่ง/ผ่อนคลาย, ฮึกเหิม/ยิ่งใหญ่
    - แต่ละอารมณ์มี hint อธิบาย เช่น "🎉 เพลงเร็ว เต้นได้ สนุกสนาน • เหมาะเปิดงาน"
  - แนวเพลง: ป๊อป, ร็อค, ดิสโก้, แร๊ฟ/ฮิปฮอป, ลูกทุ่ง/อีสาน, เพื่อชีวิต, อาร์แอนด์บี, แจ๊ส/บลูส์, เรกเก้, อินดี้
  - นักร้อง: 🎤 ชาย (pill สีฟ้า), 🎤 หญิง (pill สีชมพู), 🎤🎤 คู่ (pill สีเขียว)
- **🤖 AI Auto-Tag**: ปุ่ม AI กรอกข้อมูลให้อัตโนมัติ (ใช้ Gemini API) — แสดงเฉพาะเมื่อมีคีย์ API
- **💡 Banner เชิญชวน**: "เพิ่มจากคลังกลางได้ง่ายกว่า!" → ลิงก์ไป Select Mode ใน songs.html

### 4.2 แก้ไขเพลง (`edit-song.html`)
- ฟิลด์เหมือน add-song แต่โหลดข้อมูลเดิมมาแสดง
- รับ `?songId=xxx` จาก URL
- มีระบบ Dirty Tracking — เตือนก่อนออกจากหน้าถ้ายังไม่บันทึก
- รองรับทั้ง TH/EN ค่าจาก DB (singer: male→ชาย, female→หญิง)

## 7. Live Mode

**ไฟล์**: `live.html` | **สิทธิ์**: ทุก Role | **ธีม**: Dark Mode เต็มจอ (ไม่ใช้ Sidebar)

### หน้าจอหลัก:
- **Top Bar**: นาฬิกา, ตัวนับเพลง, ชื่อ venue, ตัวปรับขนาดตัวอักษร (A+/A-)
- **Now Playing**:
  - ชื่อเพลงขนาดใหญ่ (สีทอง #fde68a)
  - คีย์ขนาดใหญ่ (กดเพื่อ Transpose)
  - BPM, ข้อมูลนักร้อง, Notes
  - แถบเพศนักร้อง: น้ำเงิน=ชาย, ชมพู=หญิง, เขียว=คู่
  - ลูกศรเปลี่ยนเพลง (ก่อนหน้า/ถัดไป)
  - แสดง Preview เพลงถัดไป
  - Metronome (เปิด/ปิดจังหวะ, เปิด/ปิดเสียง)
  - ปุ่ม End Break
  - Pulse animation เมื่อเพลงใกล้จบ

### Song List:
- เพลงปัจจุบัน (highlight สีทอง), เพลงถัดไป (highlight สีเขียว)
- เพลงข้าม (ขีดเส้นทับ, จางลง)
- ปุ่มเลื่อนขึ้น/ลง (reorder)
- ปุ่มลบเพลง
- **Badges**: 🟣 ขอเพลง (request), 🔵 อังกอร์ (encore), ข้าม (skip), 🟡 วันเกิด (birthday)
- **Swipe Gestures**: ปัดซ้าย=ลบ, ปัดขวา=เพิ่ม

### Chat Bar (ขอเพลง):
- แถบด้านล่างคงที่ — กรอกชื่อเพลง + กดส่ง
- Autocomplete ชื่อเพลง

### อื่นๆ:
- **Context Menu**: Bottom sheet แสดง grid ปุ่ม actions
- **Gesture Hints Overlay**: Tutorial แสดงครั้งแรกสอนวิธีใช้ท่าทาง
- **Action Bar**: undo, จบเพลง, nudge
- **Modals**: Transpose, แก้ไข notes

## 24. เกมส์ดนตรี (Music Games)

**ไฟล์**: `games.html` | **สิทธิ์**: ทุก Role

### Lobby:
- **Game Grid** (5 เกม):
  1. 🎵 **ทายเพลง** — ดูคำใบ้แล้วทายว่าเพลงอะไร
  2. 🎹 **ทายคีย์** — บอกชื่อเพลง ทายว่าคีย์อะไร
  3. 🥁 **จับจังหวะ** — ฟังจังหวะแล้วเคาะตาม
  4. 🔤 **ต่อชื่อเพลง** — ต่อคำด้วยพยัญชนะตัวสุดท้าย
  5. 🎯 **ทาย BPM** — ฟัง metronome แล้วทายความเร็ว

- **แต่ละเกม**: แสดงจำนวน "กำลังเล่น" (real-time) + สถิติ
- **🟢 Online Badge**: แสดงจำนวนคนออนไลน์
- **Leaderboard**: อันดับเหรียญทอง/เงิน/ทองแดง + คะแนน
- **Online Players**: แสดงรายชื่อผู้ที่ออนไลน์อยู่

### ในเกม:
- **Game Header**: ปุ่มกลับ, ชื่อเกม, รอบที่, คะแนน
- **Timer Bar**: แถบจับเวลา (สีทองปกติ, สีแดงเมื่อเร่งด่วน)
- **Quiz/Chord**: 4 ตัวเลือก, เฉลยสีเขียว/แดง
- **Rhythm Tap**: ปุ่มกลมใหญ่สำหรับเคาะจังหวะ + beat dots + accuracy display
- **Song Chain**: แสดงตัวอักษรที่ต้องต่อ + ช่องกรอก + ประวัติเพลงที่ต่อ
- **BPM Guess**: ปุ่มฟัง metronome + ช่องกรอก BPM
- **Game Over**: คะแนนสุดท้าย + ปุ่มเล่นอีก/กลับ Lobby
- **Invite Toast**: รับ invite จากเพื่อนร่วมวง (real-time)

## 27. Ad Gate (ดูโฆษณา)

**ไฟล์**: `ad-gate.html` | **สิทธิ์**: Free tier only

- **กลไก**: ผู้ใช้ Free tier ต้องดูโฆษณา 30 วินาทีเพื่อใช้งาน 75 นาที
- **Flow**:
  1. แสดงข้อมูล: "ดูโฆษณาสั้นๆ เพื่อใช้งาน 75 นาที"
  2. กดปุ่ม "▶ เริ่มดูโฆษณา"
  3. Countdown 0:30 → 0:00
  4. เสร็จแล้ว → ปุ่ม "✅ เข้าใช้งานได้เลย" → ไป dashboard
  5. บันทึก timestamp ใน localStorage → ไม่ต้องดูอีก 75 นาที
- **ข้าม**: ลิงก์ "อัปเกรดเป็น Lite / Pro" เพื่อข้ามโฆษณาตลอด
- ผู้ใช้ Lite/Pro จะ redirect ไป dashboard ทันที

## 28. ระบบสร้างวงใหม่ (Create Band)

**ไฟล์**: `create-band.html` | **สิทธิ์**: ไม่ต้อง login

### ขั้นตอน (3 ขั้น):
1. **ข้อมูลวง**: ชื่อวง, จังหวัด (datalist ทุกจังหวัดในประเทศไทย), จำนวนสมาชิก
2. **ข้อมูลผู้จัดการ**: คำนำหน้า, ชื่อ, นามสกุล, ชื่อเล่น, ตำแหน่ง/เครื่องดนตรี (13 ตัวเลือก พร้อม "ผู้จัดการวง" เป็นค่าเริ่มต้น)
3. **สร้างบัญชี**: อีเมล, รหัสผ่าน

### Flow หลังสร้าง:
> 📖 ขั้นตอนการสร้างวงใหม่:
> 1. ผู้จัดการวงกรอกข้อมูลวง + สร้างบัญชี
> 2. รอแอดมินอนุมัติ (ปกติไม่เกิน 24 ชม.)
> 3. เมื่ออนุมัติแล้ว จะได้ **"รหัสวง"** ในหน้าหลัก
> 4. นำรหัสวงไปแจกสมาชิก เพื่อสมัครเข้าร่วมวง

## 29. ระบบ Navigation & Accessibility

**ไฟล์**: `js/nav.js` + `css/nav.css`

### Sidebar Navigation:
- เมนูแบ่ง 3 กลุ่มตาม role:
  - **🎸 สมาชิกวง** (ทุกคน)
  - **👔 ผู้จัดการวง** (manager/admin)
  - **🔧 แอดมิน** (admin only)
- เมนูย่อย (submenu) แบบ collapse/expand พร้อมลูกศร
- **Song Manager Role Injection**: สมาชิกที่อยู่ในลิสต์ Song Managers จะเห็นเมนู admin-songs โดยไม่ต้องเป็น admin

### Accessibility:
- **ขนาดตัวอักษร**: สลับ lg / xl
- **High Contrast Mode**: เปิด/ปิด
- **Simple Menu**: สลับเมนูแบบง่าย
- บันทึกค่าใน localStorage

### Design System:
- **ฟอนต์**: Kanit (หัวข้อ), Sarabun (เนื้อหา)
- **สี**: Premium Dark Gold theme (gold #f6ad55, dark #1a1a1a)
- **Touch**: ปุ่มขั้นต่ำ 44px สำหรับมือถือ
- **Responsive**: รองรับ mobile-first
- **PWA**: manifest.json, service worker, app icons, Add to Home Screen
